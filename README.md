An interactive AI Voice Assistant built using Groq AI, Murf AI, and Streamlit that allows users to ask questions and receive intelligent responses with natural human-like voice output.
This project demonstrates how modern AI models and voice technologies can be integrated to create a voice-first interactive assistant for students and general users.




Problem Statement:-
Develop an intelligent AI-powered voice assistant that enables users, especially students, to interact with technology through natural conversation by understanding queries, generating accurate AI responses, and converting them into human-like speech in real time.


Features

🤖 AI-powered responses using Groq Llama models

🔊 Realistic voice generation using Murf AI

🌊 Modern animated voice assistant UI

⏳ Voice generation status notifications

🔁 Automatic retry system for audio generation

📏 Response length optimization

🎨 Interactive and responsive interface

👨‍💻 Professional developer footer

Technologies Used:-

Technology	Purpose
Python	    -   Backend development
Streamlit   - 	Interactive UI
Groq API    -	AI response generation
Murf AI	    -  Text-to-Speech voice synthesis
